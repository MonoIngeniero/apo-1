# Ejercicios de Programación en Java (APO I)

## Descripción

Este repositorio reúne diferentes ejercicios desarrollados durante el curso **Algoritmos y Programación Orientada a Objetos I** de la Universidad Icesi.

Los ejercicios fueron realizados con el objetivo de fortalecer las bases de la programación utilizando Java, aplicando estructuras de control, arreglos, métodos y programación orientada a objetos.

---

## Contenido

Entre los ejercicios desarrollados se encuentran:

- Cálculo del promedio de notas utilizando arreglos.
- Registro y procesamiento de puntajes de jugadores.
- Prácticas con ciclos (`for` y `while`).
- Manejo de arreglos unidimensionales.
- Uso de métodos con parámetros y valores de retorno.
- Entrada de datos mediante `Scanner`.

---

## Tecnologías

- Java
- Programación Orientada a Objetos
- Eclipse / NetBeans

---

## Conceptos aplicados

- Variables y tipos de datos
- Condicionales
- Ciclos
- Métodos
- Arreglos
- Entrada y salida de datos
- Modularización del código

El propósito de este repositorio es documentar mi proceso de aprendizaje en programación y conservar los ejercicios desarrollados durante el curso.
